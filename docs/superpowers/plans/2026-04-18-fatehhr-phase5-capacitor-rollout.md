# Fateh HR — Phase 5: Capacitor Wrap + Per-Customer Build + Rollout — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the web SPA into a signed Android APK, wire per-customer theming all the way through to the adaptive icon + splash + `manifest.json`, prove the one-env-switch customer swap visibly changes the app, run `design-review` on every major screen, complete the brief's §12 verification-gate checklist, and ship `v1.0.0`.

**Architecture:** A new `android-capacitor/` directory at the repo root holds the Capacitor wrapper. A `scripts/build-customer.sh` shell script sources a per-customer `.env.<slug>` file, runs `vite build` with `CUSTOMER_BUILD_TARGET=native`, copies assets to the Android project via `cap copy android`, bumps `versionCode` + `NATIVE_VERSION`, generates the accent-tinted icon / splash / theme colour in the Android resources, and builds a signed release APK via Gradle. A fresh keystore is generated on day 1 of this phase and backed up to two locations per skill §5 rule #17.

**Tech Stack:** Capacitor 6, Android SDK 34, JDK 17, Gradle (AGP matching Capacitor 6), `sharp` (icon generation), `bash` + `gh` (scripts), manual device verification.

- [ ] **Step 2: Write the script**

```javascript
#!/usr/bin/env node
// Generate an accent-tinted adaptive icon + splash for Capacitor Android.
// Inputs (via env): CUSTOMER_PRIMARY_COLOR, CUSTOMER_BRAND_NAME
// Output: android-capacitor/android/app/src/main/res/{mipmap-*, drawable/splash.png}

import sharp from "sharp";
import fs from "node:fs/promises";
import path from "node:path";
import { fileURLToPath } from "node:url";

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const accent = (process.env.CUSTOMER_PRIMARY_COLOR || "#2E5D5A").toUpperCase();
const brand = process.env.CUSTOMER_BRAND_NAME || "Fateh HR";

const mipmapDir = path.join(__dirname, "..", "android-capacitor", "android", "app", "src", "main", "res");
const sizes = { "mipmap-mdpi": 48, "mipmap-hdpi": 72, "mipmap-xhdpi": 96, "mipmap-xxhdpi": 144, "mipmap-xxxhdpi": 192 };

async function writeSolid(pathOut, size) {
  await fs.mkdir(path.dirname(pathOut), { recursive: true });
  await sharp({
    create: { width: size, height: size, channels: 4,
              background: { r: hex(accent, 0), g: hex(accent, 1), b: hex(accent, 2), alpha: 1 } },
  }).png().toFile(pathOut);
}

function hex(color, i) {
  const c = color.replace(/^#/, "");
  return parseInt(c.slice(i * 2, i * 2 + 2), 16);
}

async function main() {
  // Simple v1 adaptive icon: solid accent bg + white khatam mark.
  // Future: composite an SVG khatam via sharp.composite.
  for (const [dir, size] of Object.entries(sizes)) {
    const outBg = path.join(mipmapDir, dir, "ic_launcher_background.png");
    await writeSolid(outBg, size);
    // Also write a square ic_launcher.png and round variant for legacy launchers
    await writeSolid(path.join(mipmapDir, dir, "ic_launcher.png"), size);
    await writeSolid(path.join(mipmapDir, dir, "ic_launcher_round.png"), size);
  }

  // Splash: solid accent, full bleed 2732x2732 (largest common target)
  const splashPath = path.join(mipmapDir, "drawable", "splash.png");
  await writeSolid(splashPath, 2732);
  console.log(`Generated adaptive icon + splash for accent ${accent} (brand: ${brand})`);
}

main().catch((e) => { console.error(e); process.exit(1); });
```

### Task 5: `bump-version.mjs` — increment versionCode + NATIVE_VERSION

**Files:**
- Create: `frontend/src/app/native-version.ts`
- Create: `scripts/bump-version.mjs`

- [ ] **Step 1: `native-version.ts`**

```typescript
// Bumped by scripts/bump-version.mjs on every release build.
// Rule (skill §5 #15): versionCode AND NATIVE_VERSION increment together.
export const NATIVE_VERSION = "1.0.0";
export const NATIVE_VERSION_CODE = 1;
```

- [ ] **Step 2: `scripts/bump-version.mjs`**

```javascript
#!/usr/bin/env node
import fs from "node:fs/promises";
import path from "node:path";
import { fileURLToPath } from "node:url";

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const target = path.join(__dirname, "..", "frontend", "src", "app", "native-version.ts");
const src = await fs.readFile(target, "utf8");
const vMatch = src.match(/NATIVE_VERSION\s*=\s*"(\d+)\.(\d+)\.(\d+)"/);
const cMatch = src.match(/NATIVE_VERSION_CODE\s*=\s*(\d+)/);
if (!vMatch || !cMatch) throw new Error("Cannot parse native-version.ts");
const kind = process.argv[2] ?? "patch"; // patch | minor | major
let [, major, minor, patch] = vMatch.map((x) => Number(x));
if (kind === "major") { major++; minor = 0; patch = 0; }
else if (kind === "minor") { minor++; patch = 0; }
else patch++;
const nextVersion = `${major}.${minor}.${patch}`;
const nextCode = Number(cMatch[1]) + 1;

const next = src
  .replace(/NATIVE_VERSION\s*=\s*"[^"]+"/, `NATIVE_VERSION = "${nextVersion}"`)
  .replace(/NATIVE_VERSION_CODE\s*=\s*\d+/, `NATIVE_VERSION_CODE = ${nextCode}`);

await fs.writeFile(target, next, "utf8");
console.log(`NATIVE_VERSION ${nextVersion} (code ${nextCode})`);
```

### Task 7: `verify-build.mjs` — assert APK matches customer env

**Files:**
- Create: `scripts/verify-build.mjs`

- [ ] **Step 1: Implement**

```javascript
#!/usr/bin/env node
// Post-build verification: open the APK, parse AndroidManifest + resources,
// and assert the app id and primary color match the expected values.
// Requires `aapt` (part of Android SDK build-tools) on PATH.

import { execFileSync } from "node:child_process";
import fs from "node:fs";

const [apk, expectedAppId, expectedColor] = process.argv.slice(2);
if (!apk || !expectedAppId || !expectedColor) {
  console.error("usage: verify-build.mjs <apk> <app_id> <#RRGGBB>"); process.exit(1);
}
if (!fs.existsSync(apk)) { console.error(`missing apk: ${apk}`); process.exit(1); }

const dump = execFileSync("aapt", ["dump", "badging", apk]).toString();
const idMatch = dump.match(/package: name='([^']+)'/);
if (!idMatch) { console.error("Cannot find package name"); process.exit(1); }
if (idMatch[1] !== expectedAppId) {
  console.error(`FAIL: app id ${idMatch[1]} !== expected ${expectedAppId}`); process.exit(1);
}

const resources = execFileSync("aapt", ["dump", "resources", apk]).toString();
const colorRe = new RegExp(`colorPrimary[\\s\\S]*?${expectedColor.toUpperCase()}`);
if (!colorRe.test(resources.toUpperCase())) {
  console.error(`FAIL: colorPrimary != ${expectedColor}`); process.exit(1);
}
console.log(`OK: ${apk}`);
```
